## Features

  

  

-  *Under Construction*
- Open Source

	**Current Features:**
	    
	**Basics:**
	    
	    
		Addition

		Subtraction

		Multiplication

		Division

	***ASMD Supports:***

		Simple (standard operations of ‚Ñù)

		Two Arrays

		Multi Dimensional Arrays

	**Geometry:**
		    
		    
		Circles

		Cylinders

		Rectangles

		Spheres
		
		Squares
		
		Triangles

	***Geometry Supports:***

		Ci,Cy,Re,Sp,Sq: Area(or surface area) & Perimeter

		Triangles: Perimeter & Area (height based, ASA, SAS, SSS)
